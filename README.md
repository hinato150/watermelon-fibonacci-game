# フィボドロップ

フィボドロップは、フィボナッチ数列をテーマにした落下合体パズルゲームです。

スイカゲームの「次にどれとどれをぶつければ大きくなるかを考える」面白さをベースに、フィボナッチ数列という数学のルールを、勉強ではなくエンタメとして自然に触れられる体験へ落とし込みました。

## Demo

[https://watermelon-fibonacci-game.vercel.app](https://watermelon-fibonacci-game.vercel.app)

## Concept

数学は、公式や暗記として出会うと少し遠く感じられることがあります。

このゲームでは、フィボナッチ数列を説明として覚えさせるのではなく、プレイヤーが遊びながら「この数字を大きくするには、どの数字とぶつければいいか」を考える構造にしています。

通常のスイカゲームでは同じ果物同士を合わせますが、フィボドロップではフィボナッチ数列で隣り合う数字を合わせます。

```text
1 + 1 = 2
1 + 2 = 3
2 + 3 = 5
3 + 5 = 8
5 + 8 = 13
8 + 13 = 21
```

このルールによって、数字の並びをただ眺めるのではなく、ゲーム上の判断としてフィボナッチ数列を使うことになります。

## Gameplay

- 画面上部から数字ブロックを落とします。
- フィボナッチ数列で合体できる数字同士が触れると、次の数字になります。
- 合体してできた数字がスコアに加算されます。
- 数字が大きくなるほどブロックも大きくなります。
- 配置済みのブロックが危険ラインを超えるとゲームオーバーです。
- 目標数字は `144` ですが、到達後もスコアを伸ばせます。

## Highlights

- フィボナッチ数列ベースの合体ルール
- スイカゲーム風の落下・衝突・積み上げ体験
- 次に合体できる相手の表示
- 到達済みの数列ガイド
- スコア、ベストスコア、最大到達数字の表示
- サイバーパンク風のネオン UI
- スマホ表示に合わせた盤面スケールとタッチ操作
- 合体時のみ発生するエフェクト
- Vercel による公開デプロイ

## Implementation

外部の物理エンジンは使わず、React と TypeScript で軽量なゲームループを実装しています。

衝突判定は、見た目に合わせて円形ベースの接触判定に寄せています。ブロック同士の重なりを押し戻し、サイズに応じた質量、反発、接触摩擦を使って、積み上がった数字が過度に揺れたり横滑りしすぎないように調整しています。

ブロックサイズは、数字が大きくなるほど視覚的に大きく見えるようにしています。

```ts
Math.min(152, 32 + value ** 0.5 * 25)
```

ゲームオーバー判定は、落下開始直後のブロックではなく、配置済みのブロックが危険ラインを超えた場合に発生します。これにより、プレイヤーが「積み上げすぎた」と直感的に分かるルールにしています。

スマホでは盤面全体を画面幅に合わせて縮小し、タップ操作と左右移動ボタンで遊べるようにしています。

## Controls

- マウス移動: 落下位置を決める
- 左クリック: ブロックを落とす
- Space: ブロックを落とす
- 左右キー / A・D: 落下位置を動かす
- R: リスタート

## Getting Started

```bash
npm install
npm run dev
```

開発サーバー起動後、表示された URL をブラウザで開きます。

```bash
npm run build
```

本番用ビルドを確認できます。

## Project Goal

この作品の狙いは、数学を「勉強するもの」としてではなく、「遊びの中で自然に理解できるもの」として見せることです。

フィボナッチ数列を題材にしながら、まずはゲームとして触って楽しいこと、その結果として数列の関係性に気づけることを大切にしています。
