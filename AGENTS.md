# AGENTS.md

## Project Overview

This is a small browser game built with Vite, React, and TypeScript.

## Commands

- Install dependencies: `npm install`
- Start development server: `npm run dev`
- Build: `npm run build`

## Project Structure

- `src/App.tsx`: Main application component
- `src/components/`: React UI components
- `src/game/`: Game logic
- `src/styles.css`: Global styles
- `PLAN.md`: Implementation plan

## Rules for Codex

- Read `PLAN.md` before making changes.
- Keep changes small and focused.
- Do not add a backend unless explicitly requested.
- Do not add external libraries unless necessary.
- After code changes, make sure the project builds.
- Prefer simple and readable code.
