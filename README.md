# フィボドロップ

フィボドロップは、フィボナッチ数列をテーマにした落下合体パズルゲームです。

スイカゲームの「次にどれとどれをぶつければ大きくなるかを考える」遊びをベースに、数学の概念であるフィボナッチ数列をエンタメとして自然に触れられる体験にすることを目指しています。

## Concept

数学は、公式や暗記として出会うと少し遠く感じられることがあります。

このゲームでは、フィボナッチ数列を説明として覚えさせるのではなく、プレイヤーが遊びながら「この数字を大きくするには、どの数字とぶつければいいか」を考える構造にしています。

たとえば、通常のスイカゲームでは同じ果物同士を合わせますが、フィボドロップではフィボナッチ数列で隣り合う数字を合わせます。

```text
1 + 1 = 2
1 + 2 = 3
2 + 3 = 5
3 + 5 = 8
5 + 8 = 13
```

このルールによって、数字の並びをただ眺めるのではなく、ゲーム上の判断としてフィボナッチ数列を使うことになります。

## Gameplay

- 画面上部から数字ブロックを落とします。
- フィボナッチ数列で合体できる数字同士が触れると、次の数字になります。
- 合体してできた数字がスコアに加算されます。
- 数字が大きくなるほどブロックも大きくなります。
- 危険ラインを超えた状態が続くとゲームオーバーです。
- 目標数字は `144` ですが、到達後もスコアを伸ばせます。

## Features

- フィボナッチ数列ベースの合体ルール
- 重力と衝突を使った落下パズル
- 次に合体できる相手の表示
- 到達済みの数列ガイド
- スコア、ベストスコア、最大到達数字の表示
- サイバーパンク風のネオン UI
- 合体時のみ発生するエフェクト

## Tech Stack

- Vite
- React
- TypeScript
- CSS

外部の物理エンジンは使わず、シンプルな矩形ベースの衝突判定と位置補正でプロトタイプしています。

## Implementation Notes

このプロトタイプでは、ゲームとしての分かりやすさを優先して、以下のような実装にしています。

- フィボナッチ数列は `1, 2, 3, 5...` として扱い、`0` は登場させません。
- `1 + 1 = 2` は序盤のテンポを出すために許可しています。
- 合体後の新しいブロックは、周囲のブロックと重ならないように生成位置を補正しています。
- 衝突時ではなく、数字が大きくなった合体時だけエフェクトを出します。
- ブロックサイズは現在 `Math.min(152, (8 + value) * 2)` で計算しています。

## Controls

- マウス移動: 落下位置を決める
- 左クリック: ブロックを落とす
- Space: ブロックを落とす
- 左右キー / A・D: 落下位置を動かす
- R: リスタート

## Getting Started

```bash
npm install
npm run dev
```

開発サーバー起動後、表示された URL をブラウザで開きます。

```bash
npm run build
```

本番用ビルドを確認できます。

## Project Goal

この作品は、数学を「勉強するもの」としてではなく、「遊びの中で自然に理解できるもの」として見せるためのプロトタイプです。

フィボナッチ数列を題材にしながら、まずはゲームとして触って楽しいこと、その結果として数列の関係性に気づけることを大切にしています。
