# PLAN

## Game Concept

小さなブラウザゲームを作る。

## Goal

プレイヤーが操作でき、スコアがあり、ゲームオーバーになるゲームを完成させる。

## Screens

- Start Screen
- Game Screen
- Game Over Screen

## Controls

- Arrow keys or WASD to move
- Space key to start or retry

## Milestones

1. Create basic screens
2. Add game state
3. Add player movement
4. Add enemies
5. Add collision detection
6. Add score
7. Add game over and retry
8. Improve UI
